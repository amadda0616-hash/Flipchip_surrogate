# Flipchip_surrogate
# 🚀 AI 기반 반도체 패키지 신뢰성 예측 프로젝트
## 반도체가 열변형에 의해 휘어지고 깨지고 뜯어지는 것을 ai로 예측합니다.
<br>

### 1. 프로젝트 개요

반도체 패키지는 서로 다른 성질의 재료들이 겹겹이 쌓인 '초정밀 샌드위치'입니다. 열을 받으면 각 재료의 팽창 속도가 달라 두 가지 치명적인 문제가 발생합니다.

뒤틀림(Warpage): 재료들이 서로 다른 방향으로 휘어지면서 전체 모양이 변형되는 현상입니다.

박리(Delamination): 휘어지는 힘을 이기지 못하고 재료 층과 층 사이가 벌어져 떨어져 나가는 현상입니다. 이는 전기적 연결을 끊어버리는 가장 치명적인 불량 사유입니다.

문제점: 이러한 불량을 확인하기 위한 기존 시뮬레이션은 한 번 수행에 너무 많은 시간과 비용이 소모됩니다.

해결: 이 뒤틀림을 예측하려면 복잡한 물리 계산(시뮬레이션)이 필요한데, 한 번 계산에 시간이 너무 오래 걸립니다. 그래서 우리는 결과를 순식간에 맞추는 AI를 가르치기로 했습니다.    
<br>                     

### 2. 대리 모델의 도입 (The "Surrogate Model" Concept)
본 프로젝트의 핵심은 복잡하고 오래걸리는 무거운 컴퓨터 시뮬레이션 작업를 대신할 **'대리 모델(Surrogate Model)'**을 구축하는 것입니다.

개념: 실제 물리 시뮬레이션은 정확하지만 너무 느립니다. 이를 대신해 입력값과 결과값 사이의 상관관계만 빠르게 계산하는 AI 모델을 만듭니다.

비유: 수학 문제를 풀 때마다 정석대로 모든 풀이 과정을 적는 대신, 수만 개의 문제 데이터를 학습해 **문제를 보자마자 답을 내놓는 '암산 천재'**를 옆에 두는 것과 같습니다.

장점: 기존 시뮬레이션으로 수 시간이 걸리던 검증을 0.1초 이내로 단축하여 실시간 설계 최적화를 가능하게 합니다.

### 4. 진행 과정 (The Process)

Step 1: 스마트한 데이터 수집 (LHS 샘플링)
무작위 추출이 아닌 수학적 기법(LHS)으로 1,200개의 최적화된 설계 조합을 생성합니다. 이는 대리 모델이 편향 없이 학습할 수 있는 양질의 교과서가 됩니다.

Step 2: 가상 시뮬레이션 및 데이터 라벨링
Ansys를 활용해 각 조합에서 발생하는 뒤틀림 정도와 각 계면의 **박리 위험도(응력 변화)**를 시계열 데이터로 추출합니다.

Step 3: 대리 모델 학습 (Chronos-2)
수집된 데이터를 시계열 AI 모델에 학습시킵니다. AI는 시간에 따른 온도 변화 속에서 어느 시점에 박리가 일어날지를 예측하는 대리 모델로 거듭납니다.

### 5. 기대 효과 (Expectation)

a. 시간 단축: 기존에 수 시간이 걸리던 설계 검증을 0.1초 만에 완료할 수 있습니다.

b. 품질 향상: 설계 단계에서 발생할 수 있는 잠재적 불량을 미리 차단하여 반도체의 수명을 늘립니다.

c. 최적화: 가장 가볍고 얇으면서도 튼튼한 **'최적의 샌드위치 레시피'**를 찾을 수 있습니다. AI가 역으로 "박리를 막으려면 이 재료의 두께를 얼마로 조절해야 한다"는 가이드를 제시할 수 있습니다.

d. 디지털 트윈 구현: 가상 환경에서 수만 번의 시행착오를 미리 거침으로써 실제 양산 시의 불량률을 0%에 가깝게 통제합니다
